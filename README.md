# HuPER: A Human-Inspired Framework for Phonetic Perception

Phone recognition model: https://huggingface.co/huper29/huper_recognizer
Phoneme-to-phone correction model: https://huggingface.co/huper29/huper_corrector

Dataset for training: https://huggingface.co/datasets/huper29/huper-clean100-proxyphones

See usage examples in the notebooks, or check the huggingface model pages.

Training code under `correction` and `wavlm_ft` directories.
